# CyberLint_AI
AI Code Guardian  – An AI-powered tool for real-time code review and bug detection. Get instant feedback, optimize performance, and improve code quality effortlessly. Supports Markdown for clear, structured insights. Debug faster, write cleaner code, and level up your development game! ⚡👨‍💻

CyberLint_Ai – AI-Powered Code Review Tool
CyberLint_Ai is an AI-driven code review tool that analyzes your code and provides instant feedback, helping developers write better, optimized, and error-free code.

Tech Stack:
Frontend: React, Markdown, rehype-highlight, Prism.js
Backend: Node.js, Express.js
AI Integration: Google Gemini API
Styling: Tailwind CSS
Version Control: Git & GitHub
How to Use:
Enter your code in the editor.
Click "Review" to get AI-powered feedback.
View suggestions and improvements in real time.
